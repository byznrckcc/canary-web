# Geliştirme Yol Haritası (Next Steps)

Projenin gelecekteki sürümlerinde sisteme eklenecek gelişmiş siber savunma özellikleri:

- [ ] **pfSense / Firewall API Entegrasyonu:** Tetiklenen `CRITICAL` seviyeli IP adreslerinin kurumsal firewall üzerinden otomatik engellenmesi (Automated Active Response).
- [ ] **WebSocket Entegrasyonu:** Dashboard ekranının yenilemeye gerek kalmadan anlık (Real-time) olarak alarmları akıtması.
- [ ] **MaxMind GeoIP2 Entegrasyonu:** Statik CIDR kontrolü yerine, yerel veritabanı üzerinden anlık ülke ve ASN tespiti.
- [ ] **Sahte AWS/SQL Token Tipleri:** Link haricinde sahte credential dosyaları üretecek yeni modüllerin eklenmesi.
